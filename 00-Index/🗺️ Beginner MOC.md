## Key Concepts to Internalize

> [!important] The Golden Mental Model
> Git doesn't track *changes* — it takes **snapshots**.
> Every commit is a complete picture of your project at that moment in time.

> [!tip] The Three States
> Every file in Git lives in one of three states:
> - **Modified** — changed but not yet staged
> - **Staged** — marked for the next commit
> - **Committed** — safely saved in Git's database

> [!note] Local First
> Almost everything in Git happens locally. You don't need internet to commit, branch, or view history.

## ⚡ Quick Command Lookup

| I want to…                       | Command                      |
| -------------------------------- | ---------------------------- |
| Start tracking a folder          | `git init`                   |
| Copy a remote repo               | `git clone <url>`            |
| See what's changed               | `git status`                 |
| Stage a file                     | `git add <file>`             |
| Stage everything                 | `git add .`                  |
| Save a snapshot                  | `git commit -m "msg"`        |
| See history                      | `git log --oneline`          |
| Create a branch                  | `git branch <name>`          |
| Create + switch to branch        | `git switch -c <name>`       |
| Switch branches                  | `git switch <name>`          |
| Merge a branch into current      | `git merge <branch>`         |
| Download + merge remote changes  | `git pull`                   |
| Upload your commits              | `git push`                   |
| Undo last commit (keep changes)  | `git reset --soft HEAD~1`    |
| Discard file changes             | `git restore <file>`         |
| Save work in progress            | `git stash`                  |
| See all branches                 | `git branch -a`              |

→ Full reference: [[⚡ Command Reference]]
